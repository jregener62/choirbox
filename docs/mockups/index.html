UI Mockups

Tippe auf ein Konzept um es anzusehen

Akkord-Eingabe A NEU Keypad-Popover
Tap auf Silbe oeffnet Popover mit strukturiertem Akkord-Keypad (A-G, ♯/♭, Modifier, Zahlen, Slash). Deterministisch, offline, ohne ML. Empfohlen fuer Phase 1.
Akkord-Eingabe B NEU Mal-Canvas
Tap auf Silbe oeffnet Canvas-Popover, User malt den Akkord. Vision-API erkennt Token und zeigt Alternativen. Auto-Advance beim Tap auf naechste Silbe.
Akkord-Eingabe C NEU Hybrid mit Tabs
Popover mit drei Tabs — Tasten / Malen / Text. User waehlt bevorzugten Modus, Einstellung wird gemerkt. Kombiniert Staerken von A und B.
Song-Export A NEU 3-Punkt-Menue + Bottom Sheet
Einstieg ueber das 3-Punkt-Menue im Song-Header. Bottom Sheet mit "Teilen" (OS-Share) und "Als .songbox exportieren" neben den bestehenden Aktionen (Umbenennen, Loeschen).
Song-Export B NEU Export-Modal mit Paket-Preview
Zentriertes Modal zeigt den Paket-Inhalt vor dem Export: Audio, Dokumente, Chordsheets, Sections — inkl. Gesamtgroesse. Hinweis "Keine User-Daten". Primaer-Button "Teilen" (OS-Share) plus Sekundaer "Speichern".
Song-Import C NEU Import-Flow mit Vorschau
Gegenstueck: Admin laedt .songbox hoch, sieht Quelle + Inhalte, waehlt Ziel-Ordner. Drop-Zone, Content-Grid (Audio/Docs/Chords/Sections), Konflikt-Hinweis bei Namensgleichheit.
Ansichtsmodus A NEU Toggle Song/Texte
Mode-Toggle im Header: Song-Modus (alle Ordner + Audio) vs. Texte-Modus (nur Texte/Chord-Sheets, kein Audio). Zwei Szenen zum Vergleich.
Ansichtsmodus B NEU Texte-Root: Direkt zum Viewer
Texte-Modus im Detail: Song antippen oeffnet direkt den Chord-Sheet-Viewer — kein Umweg ueber Song-Ordner. Tab-Leiste bei mehreren Dokumenten. Gast-Header mit Logout + Uhrzeit.
Ansichtsmodus C NEU Guest-Link + Ansichtswahl
Admin-UI: Gast-Link erstellen mit Auswahl "Gaeste sehen: Alles (Songs + Audio) | Nur Texte". Radio-Cards mit Beschreibung. Ergebnis-Banner zeigt gewaehlten Modus.
Ansichtsmodus D NEU Eigene View-Bar
Eigene Top-Bar oberhalb der bestehenden Toolbar. Nur sichtbar fuer Members+ (beide Ansichten freigeschaltet). Verschwindet im .song-Ordner. 4 Szenen: Songs-Root, Texte-Root, Im Song, Gast ohne View-Bar.
Song-Kachel A NEU Segmented Solid Fill
Multi-Button als iOS-Segmented-Control. Aktives Segment: solide Fuellung in Typ-Farbe. Kachel nicht mehr klickbar.
Song-Kachel B NEU Underline Active
Multi-Button als Tab-Leiste. Aktiver Button: farbige Underline wie im Text-Groesse-Selector. Kachel nicht mehr klickbar.
Song-Kachel C NEU Outlined + Glow
Multi-Button behaelt das heutige Outline-Look, jeder Button mit eigener Typ-Farbe. Aktiv: kraeftigere Fuellung + Glow.
Autoscroll im Vollbild NEU
3 Varianten fuer die Geschwindigkeitseinstellung im Vollbildmodus — Stepper, vertikaler Slider, Bottom-Sheet
