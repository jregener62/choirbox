Blättermodus — Variante B (Edge-Tap + Swipe)

Minimalistisch, maximale Sicht auf das Chord-Sheet. Keine feste Bottom-Bar. Navigation per Swipe oder Tap auf die Ränder. Status-Pill oben erscheint nur kurz beim Seitenwechsel. Ziel: Konzert-taugliches Lesegefühl wie in einer E-Reader-App.
