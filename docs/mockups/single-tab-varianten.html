Vorschlag B
Bottom Indicator — Unterstrich am aktiven Badge
Die Song Card mit Badges, aber der aktive Badge bekommt einen farbigen Unterstrich (wie ein Tab-Indicator). Die Badge-Form bleibt gleich, der Unterstrich signalisiert: "Du bist hier". Kein extra Glow noetig.

Vorschlag C
Expanded Label — aktiver Badge zeigt den Namen
Der aktive Badge wird erweitert und zeigt den Typ-Namen (z.B. "Texte", "Audio"). Die inaktiven Badges bleiben kompakt mit nur Icon + Zahl. Das Label macht sofort klar, welcher Bereich gerade angezeigt wird.

Vergleich
A — Filled Glow
+ Staerkster visueller Kontrast zum Normal-Badge
+ Subtiler Glow gibt "aktiv"-Gefuehl
+ Inaktive Badges gedimmt — klare Hierarchie
− Glow koennte auf OLED-Screens auffaellig wirken
B — Bottom Indicator
+ Etabliertes Tab-Pattern (Browser-Tabs, Material)
+ Minimale Aenderung — Badge-Form bleibt 100% gleich
+ Inaktive Badges behalten volle Sichtbarkeit
− Unterstrich ist auf kleinen Badges subtil
C — Expanded Label
+ Textlabel macht den aktiven Typ sofort klar
+ Groessenunterschied aktiv/inaktiv schafft Hierarchie
− Badge-Breite aendert sich beim Wechsel (Layout-Shift)
